# 플로이드-워셜 알고리즘 (Floyd-Warshall Algorithm)

## 개요
- 목적: 가중 그래프에서 모든 정점 쌍 간의 최단 경로 계산
- 유형: 동적 계획법(Dynamic Programming)

## 특징
- 모든 정점 쌍의 최단 경로: 각 정점 쌍 간에 가능한 모든 경로를 고려
- 가중치 포함: 음수 가중치가 있는 간선도 처리 가능
- 음의 사이클 처리 불가: 음의 사이클이 있으면 적용 불가
- 시간 복잡도: O(N^3) (N은 정점의 수)


## 알고리즘 개념
- 모든 정점 쌍 (i, j)에 대해, 모든 중간 정점 k를 거쳐가는 경로를 고려
- 각 단계에서 i에서 j로 가는 최단 경로는 i에서 k를 거쳐 j로 가는 경로와 비교하여 갱신
